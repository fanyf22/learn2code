# 前端代码使用的语言

理论上服务器对不同计算机上的不同浏览器发出的请求作出同样的响应。

因此，前端代码必须使用统一的语言，能够被所有浏览器理解。

事实上，前端代码使用 `HTML+CSS+JavaScript` 这三个语言来实现页面交互。

- **HTML** (HyperText Markup Language) 叫做超文本标记语言，用来表示页面中有哪些元素。

- **CSS** (Cascading Style Sheets) 叫做层叠样式表，用来描述页面中元素的呈现方式和排版。

- **JavaScript** 是一种解释型语言，在网页中用来处理各种动态事件和交互逻辑。

在之后的网页编程教程中我们会详细讨论这三个语言。
