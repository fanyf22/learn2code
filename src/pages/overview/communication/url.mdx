# URL

URL 类似于文件路径，是一种用于定位互联网上资源的地址。

URL 是一行纯文本，包含了协议名称、服务器域名或 IP 地址、端口、请求路径这些信息。

![URL Example](/images/url-example.svg)

- **协议**：可选，默认为 `http://`。协议名称，以 `://` 结尾。通常是 `http://` 或 `https://`。
- **服务器名称**：要访问的服务器在网络上的名称，通常是其域名或者 IP 地址。
- **端口**：可选，默认为 `80` 端口。向服务器的对应端口发送请求。
- **请求路径**：可选，默认为 `/`。请求资源的路径。
