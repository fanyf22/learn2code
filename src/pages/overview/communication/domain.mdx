# 域名

域名是互联网上作为网站地址的易于记忆的名称，用于代替复杂的 IP 地址。

每个域名为一系列字母、数字以及连字符 `-` 由 `.` 连接而成，例如 `learn2code.topfyf.cn`。

每个域名在全球范围内唯一，可以解析为唯一的 IP 地址。

将域名用 `.` 从右到左进行分级，最右边称为顶级域名，第二个称为二级域名，以此类推。

## 顶级域名

顶级域名不能单独作为域名使用，至少得有二级域名才能解析为 IP 地址。

顶级域名分为三类：

- **国家或地区**：例如中国是 `.cn`，日本是 `.jp` 等。
- **通用顶级域名**：例如 `.com` 代表商业，`.net` 代表网络提供商，`.org` 代表非营利组织等。
- **新顶级域名**：例如通用的 `.xyz`、代表高端的 `.top`、代表红色的 `.red` 等。

## 二级域名

二级域名是最高的可以解析为 IP 地址的层级。

个人或组织申请域名实际上就是申请二级域名。

例如 `learn2code.topfyf.cn` 二级域名就是 `topfyf.cn`。

每个二级域名都可以添加若干与之相关的记录，用来保存相关信息。

所有二级域名的信息都被保存在分布在世界各地的 DNS 服务器上。

浏览器在访问未知的二级域名时都会向 DNS 服务器询问对应的 IP 地址。

## 次级域名

三级及以上的域名都可以称为次级域名。

这一类的域名都视作对应二级域名的子域名。

次级域名对应的 IP 地址实际上是作为二级域名的一条记录保存的。

例如 `learn2code.topfyf.cn` 的 IP 地址实际上就是 `topfyf.cn` 的一条名为 `learn2code` 的记录。

因此如果你拥有了一个二级域名，那么你就拥有了所有次级域名。
